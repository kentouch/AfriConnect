# BestRenties

BestRenties is a modern, interactive web platform designed to serve as a central hub for real estate, specialized land services, and tourism information across key African countries, starting with Senegal, Burundi, Rwanda, and Gabon.

## Features

- **Real Estate Listings**: Browse and search for properties across various categories and locations.
- **Land Services**: Access specialized services related to land management and transactions.
- **Tourism Information**: Discover tourist sites, attractions, and travel information for key African countries.

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/bestrenties.git
   cd bestrenties
   ```

2. Create a virtual environment:
   ```
   python -m venv venv
   source venv/bin/activate  # On Windows use `venv\Scripts\activate`
   ```

3. Install the required packages:
   ```
   pip install -r requirements.txt
   ```

4. Apply migrations:
   ```
   python manage.py migrate
   ```

5. Run the development server:
   ```
   python manage.py runserver
   ```

## Usage

- Access the application at `http://127.0.0.1:8000/`.
- Explore the different sections for real estate, land services, and tourism.
