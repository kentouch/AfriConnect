1. Project overview

AfriPropConnect aims to make property discovery and management simple for users and agents. The platform supports:

Residential and commercial rentals

Property sales (houses, apartments)

Land services: land listings, plot maps, surveying/ownership documents, land development services

Search & filter (location, price, property type, size)

Saved searches, favourites, notifications

Agent dashboards and admin moderation

2. Key features

Public listing pages with photos, map, amenities and contact form

Advanced search: by location (city/district), price range, bedrooms, land size, property type

Multi-purpose listing types: rental, sale, land (each with type-specific fields)

Authenticated flows: signup/login, profile, agent verification

Listing management: create/edit/publish/unpublish, gallery uploader, availability calendar
